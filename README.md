e-commmerce-data-insights
Analysing 100k Brazilian e-commerce orders with SQL, Excel &amp; Power BI to uncover why customers churn and what late deliveries cost.
An end-to-end data analysis of ~100,000 real orders from Olist, Brazil's largest e-commerce marketplace (2016–2018). Built with SQL(querying & cleaning), Excel (RFM segmentation), and Power BI.


Tools:,SQL,Excel,Power BI
Dataset:[Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)
